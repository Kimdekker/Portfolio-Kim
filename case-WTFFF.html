WTFFF!?

WTFFF!? is an educative website about online sexual abuse, made for fonds slachtofferhulp

Challenge

The teams challenge was making a portal between the different sites of content, so it allows for easy navigation between the URL's. It was particularly challenging to make this portal, because the different sites and URL's used different frameworks, and different codebases. The solution was creating a portal over all URL's functioning as a navigation. It looks like a normal navigation, but under the hood it's a bit more complicated. But here we are!

My specific challenge was to work with a team of developers using Git and working with React, Next, Typescript and Tailwind CSS for the first time in a (professional) project.

My part

I worked on making the components in React, working together with the UI designer to make the final design look as accurately as possible, while keeping in mind accessibility and keeping the original look the same as it does on the other page. For example, the hover effects on the buttons, the accordion and dropdowns, and the headings etc. But then in another framework.
